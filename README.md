# Portfolio — Maroua KOURDI

Site web personnel présentant mon profil de développeuse Full Stack et designer UI/UX.

## Aperçu
Portfolio responsive développé avec **HTML**, **CSS**,  
Permet de présenter : accueil, à propos, compétences, projets et un formulaire de contact.

## Pages / Sections
- **Accueil** : Photo, nom, titre, courts liens (Email, LinkedIn, GitHub, CV).  
- **À propos** : Présentation personnelle, langues, compétences techniques.  
- **Projets** : Cartes de projets avec image et description courte.  
- **Contact** : Formulaire pour envoyer un message (frontend).

## Technologies
- HTML5
- CSS3 (responsive + media queries)

## Comment lancer en local
1. Ouvrir `index.html` dans un navigateur (double‑clic).  
   - Avec VS Code : utiliser l'extension "Live Server".
